        • 反证法 (Proof by contradiction)
          • 普通形式的反证法，即通过反证法证明一个结论 PPP 为真
            • 原理
              • 设结论为假，反向推导出矛盾，以否定这个结论
              • 真值表上 PPP 的真值和 ¬P  ⟹  C∧¬C\lnot P\implies{C\land{\lnot C}}¬P⟹C∧¬C 相同
                • 因为 P  ⟹  QP\implies{Q}P⟹Q 可以翻译为 ¬P∨Q\lnot P\lor Q¬P∨Q
                • 因此 ¬P  ⟹  C∧¬C\lnot P\implies{C\land\lnot C}¬P⟹C∧¬C 可以翻译为 P∨(C∧¬C){P}\lor({C\land{\lnot{C}}})P∨(C∧¬C)
                • 由于 C∧¬CC\land\lnot{C}C∧¬C 永远为 False，因此该项的真值就取决于 PPP的真值
                • 因此要证明 PPP 和证明 ¬P  ⟹  C∧¬C\lnot P\implies{C\land\lnot{C}}¬P⟹C∧¬C 相同
            • 方法：首先写出 ¬P\lnot P¬P，然后逐步推导出 C∧¬CC\land\lnot{C}C∧¬C。¬P\lnot P¬P 时第一步，C∧¬CC\land\lnot{C}C∧¬C 是最后一步。CCC 可能不是命题中的一部分，是某个可以推导出的矛盾点。

          • Conditional Statement （即 if P, then Q 形式）的反证法证明
            • 原理
              • 需要注意的是，反证否定整个结论。在之前我们的结论是 PPP，因此否定后的结论是 ¬P\lnot P¬P。现在我们 P  ⟹  QP\implies{Q}P⟹Q，那么否定后的结论就是 ¬(P  ⟹  Q)\lnot({P\implies{Q}})¬(P⟹Q)
              • 我们的目的是从 ¬(P  ⟹  Q)\lnot({P\implies{Q}})¬(P⟹Q) 为 True 开始证，也就是说 P  ⟹  QP\implies{Q}P⟹Q 为 False
              • 分析 P  ⟹  QP\implies{Q}P⟹Q，我们知道只有在 PPP 为 True 且 QQQ 为 False 时结论为 False。那么就从这里开始证，也就是第一步设定 PPP 且 ¬Q\lnot{Q}¬Q。最后推导出一个矛盾。
